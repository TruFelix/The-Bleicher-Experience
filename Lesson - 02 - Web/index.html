Navigation-Bar

A Navigation-Bar enables the user to jump around the page with ease. This is useful for the user as it enables him to quickly find the issue that he is looking for.

CSS-Grid

CSS-Grid is a relatively new technology for CSS. It is a very powerful tool, as you will find out shortly.

You can use grid to create predictable behaviour when dealing with 2d arangements, though it also can be used to create 1d arrangements.

  1. To enable grid just do display: grid

  2. And set grid-template-column and grid-template-row

    These specify the size of each colomn and row. The unit to use is generaly fr.

    A fr represents one fraction.

    So if you have grid-template-column: 1fr 1fr 3fr; 1fr is 1/5th of the width, 3fr is 3/5ths of the whole width.
